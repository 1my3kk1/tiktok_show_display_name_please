### cách sử dụng

#### extension:
1. tải file code và giải nén
2. mở chrome://extensions/
3. chọn tải "tiện ích đã giải nén" (nếu không có thì bật "chế độ dành cho nhà phát triển" góc trên bên phải)
4. chọn thư mục đã giải nén

#### userscript:

1. tải extension [![Tampermonkey](https://img.shields.io/badge/Tampermonkey--gray?logo=tampermonkey)](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
2. tạo **tập lệnh mới** trong tampermonkey
3. dán toàn bộ nội dung của file [tiktok_show_display_name_please.user.js](tiktok_show_display_name_please.user.js) vào
4. `ctrl (or cmd) + s` để lưu
